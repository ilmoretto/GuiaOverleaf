Módulo 4: Elementos Flutuantes: Figuras e Tabelas

Documentos técnicos e acadêmicos raramente contêm apenas texto. Aprender a incorporar figuras e tabelas é um passo essencial. No LaTeX, esses elementos são tratados como "flutuantes", pois o LaTeX pode movê-los para um local ótimo na página para evitar quebras de página ruins e manter um bom layout.

Inserindo Figuras

Para inserir imagens, você precisa do pacote graphicx. O processo envolve três comandos principais dentro do ambiente figure: \includegraphics para incluir o arquivo da imagem, \caption para a legenda e \label para a referência cruzada.

Código LaTeX

\usepackage{graphicx}
% ...
Como vemos na Figura~\ref{fig:logo}, o logotipo é simples.
\begin{figure}[ht]
    \centering
    % Use um placeholder se não tiver a imagem
    \includegraphics[width=0.5\textwidth]{placeholder.png} 
    \caption{Logotipo da Empresa.}
    \label{fig:logo}
\end{figure}

Construindo Tabelas Profissionais

Tabelas são criadas com o ambiente tabular. Para um visual mais limpo e profissional, é altamente recomendável usar o pacote booktabs, que fornece linhas horizontais com espaçamentos melhores (\toprule, \midrule, \bottomrule) e desencoraja o uso de linhas verticais.

Código LaTeX

\usepackage{booktabs}
%...
A Tabela~\ref{tab:dados} mostra os resultados.
\begin{table}[ht]
    \centering
    \caption{Resultados da Simulação.}
    \label{tab:dados}
    \begin{tabular}{l c r}
        \toprule
        Parâmetro & Símbolo & Valor Médio \\
        \midrule
        Velocidade & $v$ & 25.4 m/s \\
        Aceleração & $a$ & 9.8 m/s$^2$ \\
        \bottomrule
    \end{tabular}
\end{table}

Proposta de Exercícios

  1. Encontre uma imagem simples online (um gráfico, um diagrama) e faça o upload para o seu projeto Overleaf.
  2. Use o ambiente figure para inserir esta imagem no seu documento. Adicione uma legenda e um \label.
  3. Escreva um parágrafo que referencie a figura usando \ref.
  4. Crie uma tabela simples com 3 colunas (ex: "Produto", "Quantidade", "Preço") e 3 linhas de dados, usando as boas práticas do booktabs.
  5. Adicione uma legenda e um \label à sua tabela e referencie-a no texto.
