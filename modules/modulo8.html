Módulo 8: Projeto Final - Seu Primeiro Artigo Acadêmico

Chegou a hora de consolidar todo o conhecimento adquirido em um projeto prático. O objetivo é criar um pequeno artigo acadêmico (de 3 a 5 páginas), do zero, utilizando a classe abntex2 e aplicando as melhores práticas que aprendemos ao longo deste guia.

Briefing do Projeto

Você irá criar um documento sobre um tema de sua escolha (pode ser um resumo de um filme, um artigo sobre seu hobby, uma breve revisão de um tópico que você domina). O documento deve ter a seguinte estrutura:

  • Estrutura ABNT Completa: Capa, Folha de Rosto, Resumo, Palavras-Chave e Sumário devem ser gerados automaticamente.
  • Conteúdo Organizado: O corpo textual deve ser dividido em, no mínimo, três capítulos: Introdução, Desenvolvimento e Conclusão.
  • Conteúdo Rico: O capítulo de Desenvolvimento deve conter obrigatoriamente:
    • Pelo menos uma figura com legenda, label e referência no texto.
    • Pelo menos uma tabela com legenda, label e referência no texto.
    • Pelo menos uma equação numerada com label e referência no texto.
  • Referências: A bibliografia deve ser gerenciada por um arquivo .bib externo com pelo menos 3 entradas. Todas as entradas devem ser citadas no texto.
  • Código Limpo: O projeto deve ser organizado em múltiplos arquivos, com um main.tex principal e arquivos separados para cada capítulo.

Estrutura de Arquivos Sugerida

No seu projeto Overleaf, crie a seguinte estrutura de arquivos:

  • main.tex (o arquivo principal)
  • cap-introducao.tex
  • cap-desenvolvimento.tex
  • cap-conclusao.tex
  • bibliografia.bib

Código LaTeX (Template para main.tex)

\documentclass[12pt, a4paper, oneside]{abntex2}
% --- Pacotes Essenciais ---
\usepackage[brazil]{babel}
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage[alf]{abntex2cite}
\usepackage{graphicx}
\usepackage{booktabs}
\usepackage{amsmath}

% --- Informações do Trabalho (Preencha com seus dados) ---
\titulo{O Título do Seu Artigo}
\autor{Seu Nome}
\instituicao{Sua Instituição}
% ... complete as outras informações ...
\local{Sua Cidade}
\data{2025}

\begin{document}
\imprimircapa
\imprimirfolhaderosto

\begin{resumo}
% Escreva seu resumo aqui...
\palavraschave{Palavra 1. Palavra 2. Palavra 3.}
\end{resumo}

\tableofcontents

% --- Corpo do Trabalho ---
\textual
\input{cap-introducao.tex}
\input{cap-desenvolvimento.tex}
\input{cap-conclusao.tex}

% --- Referências ---
\postextual
\bibliography{bibliografia}

\end{document}

Passos para a Execução

  1. Crie a estrutura de arquivos sugerida no seu projeto Overleaf.
  2. Copie o template acima para o seu arquivo main.tex e preencha suas informações.
  3. Desenvolva o conteúdo de cada capítulo nos arquivos .tex correspondentes. Lembre-se de cumprir os requisitos de conteúdo rico para o capítulo de desenvolvimento.
  4. Popule seu arquivo bibliografia.bib com pelo menos três fontes e cite-as ao longo do seu texto.
  5. Compile seu projeto frequentemente para corrigir erros à medida que aparecem.
  6. Parabéns! Ao concluir, você terá criado um documento acadêmico completo e bem estruturado, demonstrando seu domínio sobre os principais recursos do LaTeX.
