Módulo 1: O Documento Essencial

Todo documento em LaTeX possui uma estrutura mínima obrigatória. Vamos dissecar o código de um "Olá, Mundo!" para entender as partes fundamentais: o preâmbulo e o corpo do documento.

Anatomia de um Arquivo .tex

O código abaixo é o mínimo necessário para gerar um documento válido. Copie-o, cole em um novo projeto no Overleaf e clique em "Recompile" para ver a mágica acontecer.

Código LaTeX

\documentclass[12pt]{article}

\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage[brazil]{babel}

\title{Meu Primeiro Documento}
\author{Seu Nome}
\date{\today}

\begin{document}
    \maketitle
    
    Olá, mundo LaTeX! Este é meu primeiro parágrafo.
    
    Parágrafos são separados por uma linha em branco no código.
\end{document}

Desvendando o Código

  • \documentclass[12pt]{article}: A primeira linha de todo documento. Define o tipo (classe) do documento. article é ideal para artigos, relatórios curtos. Outras classes comuns são report, book e beamer (para apresentações). As opções entre colchetes, como [12pt], ajustam configurações globais, neste caso, o tamanho da fonte.
  • O Preâmbulo: É tudo que vem entre o \documentclass e o \begin{document}. Aqui, você carrega pacotes (funcionalidades extras) e define configurações globais.
    • \usepackage{...}: Carrega um pacote. inputenc com a opção utf8 permite que você use acentos diretamente no código. babel com a opção brazil ensina ao LaTeX as regras de hifenização e os termos em português (como "Sumário" em vez de "Contents").
    • \title{...}, \author{...}, \date{...}: Comandos que armazenam informações para serem usadas posteriormente. \today é um comando especial que insere a data atual.
  • O Corpo do Documento: É o conteúdo que vai entre \begin{document} e \end{document}.
    • \maketitle: Comando que pega as informações de título, autor e data definidas no preâmbulo e as formata no topo do documento.
    • Texto e Parágrafos: O texto que você escreve aparece diretamente no PDF. Para iniciar um novo parágrafo, basta deixar uma linha em branco no seu código-fonte.

Proposta de Exercícios

  1. Crie um novo projeto no Overleaf chamado "Meus Primeiros Passos".
  2. Altere o autor para o seu nome completo e o título para "Relatório de Aprendizagem".
  3. Troque \today por uma data específica, como \date{Dezembro de 2025}.
  4. Escreva três parágrafos curtos sobre o que você espera aprender com o LaTeX. Lembre-se de separar cada parágrafo com uma linha em branco no código.
  5. Compile o documento e veja o resultado no PDF.
