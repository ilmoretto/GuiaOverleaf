Módulo 5: Gerenciamento de Bibliografia com BibTeX

Gerenciar dezenas ou centenas de referências manualmente é uma tarefa propensa a erros e extremamente tediosa. O BibTeX é o sistema clássico do LaTeX para automatizar completamente a criação de citações e da lista de referências, garantindo consistência e formatação correta com o mínimo de esforço.

O Fluxo de Trabalho com BibTeX

O processo se baseia em separar os dados da apresentação:

  1. Crie um banco de dados (arquivo .bib): Você cria um arquivo de texto simples (ex: referencias.bib) onde cada entrada é uma fonte (livro, artigo, site) formatada de uma maneira específica.
  2. Cite no seu texto (arquivo .tex): No seu documento principal, você usa o comando \cite{chave_da_entrada} para citar uma fonte. A "chave" é um apelido único que você deu para a entrada no arquivo .bib.
  3. Deixe o LaTeX fazer o trabalho: Você informa ao LaTeX qual estilo de citação usar (ex: ABNT, APA, IEEE) e onde inserir a lista de referências. O LaTeX irá automaticamente buscar as informações no arquivo .bib apenas das fontes que você citou, formatá-las no estilo correto e gerar a lista completa.

Anatomia de um Arquivo .bib

No Overleaf, clique em "New File" e nomeie-o como referencias.bib. Cada entrada começa com @tipodocumento (ex: @book, @article, @misc para sites), seguido por uma chave de citação única (ex: knuth1984) e os campos de informação.

Código (arquivo referencias.bib)

@book{knuth1984,
  author    = {Donald E. Knuth},
  title     = {The TeXbook},
  publisher = {Addison-Wesley},
  year      = {1984},
  address   = {Reading, MA}
}

@article{einstein1905,
  author  = {Albert Einstein},
  title   = {Zur Elektrodynamik bewegter Körper},
  journal = {Annalen der Physik},
  volume  = {322},
  number  = {10},
  pages   = {891--921},
  year    = {1905}
}

Usando a Bibliografia no Documento Principal

No seu arquivo .tex, você precisa de três coisas: o comando \cite{} para fazer a citação, \bibliographystyle{} para definir o estilo e \bibliography{} para dizer qual arquivo .bib usar e onde imprimir a lista de referências.

Código (arquivo main.tex) 

\documentclass{article}
\usepackage[brazil]{babel}
% ...
\begin{document}
A tipografia digital foi revolucionada por Knuth~\cite{knuth1984}.
As bases da relatividade foram lançadas por Einstein~\cite{einstein1905}.

\bibliographystyle{plain} % Estilo de citação numérico simples
\bibliography{referencias} % Nome do arquivo .bib (sem a extensão)
\end{document}

Visualização do Resultado

A tipografia digital foi revolucionada por Knuth [2]. As bases da relatividade foram lançadas por Einstein [1].

Referências

[1] Albert Einstein. Zur Elektrodynamik bewegter Körper. Annalen der Physik, 322(10):891–921, 1905.

[2] Donald E. Knuth. The TeXbook. Addison-Wesley, Reading, MA, 1984.

Proposta de Exercícios

  1. Crie um novo arquivo bibliografia.bib no seu projeto.
  2. Procure no Google Scholar por um artigo do seu interesse e clique em "Citar" (ou aspas). Selecione a opção "BibTeX" e copie o código para o seu arquivo.
  3. Adicione uma entrada do tipo @book para um livro que você goste.
  4. No seu arquivo .tex principal, escreva dois parágrafos, cada um citando uma das fontes que você adicionou.
  5. Adicione os comandos \bibliographystyle{unsrt} (que ordena as referências pela ordem de citação) e \bibliography{bibliografia} no final do seu documento e recompile para ver a lista de referências ser gerada.
