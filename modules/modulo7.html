Módulo 7: Customização e Escalando Projetos

À medida que seus documentos se tornam mais complexos, você perceberá que repete os mesmos blocos de código ou formatações. O LaTeX permite criar seus próprios comandos e ambientes para automatizar essas tarefas, tornando seu código mais limpo, legível e fácil de manter. Além disso, vamos ver como organizar projetos grandes em múltiplos arquivos.

Criando Seus Próprios Comandos com \newcommand

O comando \newcommand{\nome_do_comando}[num_args]{definicao} permite criar atalhos. Isso é útil para termos técnicos, notações matemáticas ou qualquer formatação que você usa repetidamente.

Código LaTeX

\usepackage{amsfonts} % para \mathbb
% --- Comandos definidos no preâmbulo ---
\newcommand{\R}{\mathbb{R}} % Comando sem argumentos
\newcommand{\keyword}[1]{\texttt{#1}} % Comando com 1 argumento

% --- Uso no corpo do documento ---
O conjunto dos números reais é denotado por $\R$.
A palavra-chave \keyword{for} é usada para laços de repetição.

Visualização do Resultado

Criando Seus Próprios Ambientes com \newenvironment

Para blocos de conteúdo com uma estrutura de início e fim, como caixas de destaque, exemplos ou exercícios, você pode criar um ambiente. A sintaxe é \newenvironment{nome}{código_inicial}{código_final}.

Código LaTeX

% --- Ambiente definido no preâmbulo ---
\newenvironment{destaque}
    {\begin{quote}\sffamily\bfseries}
    {\end{quote}}

% --- Uso no corpo do documento ---
\begin{destaque}
    Esta é uma informação importante que precisa de destaque
    visual no documento.
\end{destaque}

Organizando Documentos Grandes com \input e \include

Escrever uma tese inteira em um único arquivo .tex é inviável. A melhor prática é dividir o trabalho em arquivos menores (um para cada capítulo, por exemplo) e juntá-los no arquivo principal.

  • \input{capitulo1.tex}: Age como um "copia e cola". O conteúdo do arquivo capitulo1.tex é inserido exatamente naquele ponto. É simples e eficaz para arquivos menores.
  • \include{capitulo2}: É mais robusto. Ele sempre inicia uma nova página antes de inserir o conteúdo e é mais inteligente para o processo de compilação. Para documentos muito grandes, você pode usar \includeonly{capitulo2} no preâmbulo para compilar apenas aquele capítulo, economizando muito tempo.

Proposta de Exercícios

  1. Crie um \newcommand chamado \latex que produza a palavra "LaTeX" com a formatação correta. (Dica: o comando original é \LaTeX).
  2. Crie um novo ambiente chamado exercicio que coloque o texto dentro de uma caixa e em itálico. (Dica: pesquise o ambiente quote e o comando \itshape).
  3. Crie dois novos arquivos no seu projeto: introducao.tex e desenvolvimento.tex.
  4. Mova o conteúdo das suas seções de introdução e desenvolvimento do arquivo principal para esses novos arquivos.
  5. Use o comando \input{} no arquivo principal para incluir o conteúdo dos novos arquivos e recompile para garantir que tudo funcione.
