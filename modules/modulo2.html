Módulo 2: Formatando Conteúdo e Estrutura

Agora que você sabe criar um documento básico, vamos aprender a organizar o conteúdo de forma hierárquica e a aplicar formatações essenciais como negrito, itálico e listas.

Hierarquia com Seções

Em vez de formatar títulos manualmente, você informa ao LaTeX a estrutura do seu documento. Ele cuidará da numeração, do tamanho da fonte e da inserção no sumário (que veremos mais à frente).

Código LaTeX

\section{Introdução}
Este é o primeiro capítulo do nosso documento.

\subsection{Motivação}
Aqui explicamos o porquê deste estudo.

\subsubsection{Objetivos Específicos}
Um nível de detalhe ainda maior.

\section{Desenvolvimento}
Iniciamos um novo capítulo, e a numeração continua automaticamente.

Ênfase e Estilos de Fonte

A forma correta de destacar um texto é usando o comando \emph{...} (ênfase). Ele geralmente resulta em itálico, mas tem um comportamento inteligente: se o texto ao redor já estiver em itálico, a ênfase será em texto reto, para criar contraste. Use \textbf{...} para negrito e \textit{...} para itálico quando a razão for puramente estilística.

Criando Listas

Listas são criadas com "ambientes". Um ambiente começa com \begin{nome_do_ambiente} e termina com \end{nome_do_ambiente}. Os itens da lista são marcados com \item.

Código LaTeX

\subsection{Tipos de Listas}
Uma lista de marcadores:
\begin{itemize}
    \item Primeiro ponto.
    \item Segundo ponto.
\end{itemize}

Uma lista numerada:
\begin{enumerate}
    \item Primeiro passo.
    \item Segundo passo.
    \item Terceiro passo.
\end{enumerate}

Proposta de Exercícios

  1. No seu projeto, crie uma seção chamada "Resumo do Módulo 2".
  2. Dentro dela, crie duas subseções: "Principais Comandos" e "Dificuldades Encontradas".
  3. Na primeira subseção, use uma lista itemize para listar os comandos que você aprendeu neste módulo (\section, \textbf, etc.).
  4. Na segunda subseção, use uma lista enumerate para descrever, em ordem, os passos que você daria para resolver um erro de compilação.
  5. Use \textbf para destacar o nome de cada comando na sua lista e \emph em uma frase na sua segunda subseção.
