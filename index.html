Módulo 0: A Filosofia do LaTeX e o Ambiente Overleaf

Antes de escrever o código, é crucial entender o "porquê". Este módulo apresenta os conceitos fundamentais que diferenciam o LaTeX de outros editores de texto e como o Overleaf facilita seu uso.

1. Foco no Conteúdo

Com o LaTeX, você se concentra na escrita e na estrutura lógica do seu texto. A formatação é tratada separadamente, garantindo consistência visual.

2. Qualidade Tipográfica Superior

O LaTeX utiliza algoritmos avançados de quebra de linha e justificação, produzindo documentos com uma qualidade visual profissional.

3. Estabilidade e Longevidade

Documentos em formato .tex são arquivos de texto puro, independentes de plataforma e acessíveis por décadas.

4. Ecossistema Poderoso

O LaTeX é sustentado por uma vasta comunidade e um repositório central de pacotes (CTAN) para qualquer necessidade de formatação.

Módulo 1: O Documento Essencial

Todo documento LaTeX possui uma estrutura fundamental. Aprenda sobre o preâmbulo, o corpo do documento e as classes que definem seu tipo.

Estrutura Mínima e Opções da Classe

O comando \documentclass pode receber opções que alteram o layout global, como tamanho da fonte (12pt) e formato do papel (a4paper).

\documentclass[12pt, a4paper, twoside]{book}

% --- PREÂMBULO COMEÇA AQUI ---
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage[brazil]{babel}
% ... outros pacotes ...
% --- PREÂMBULO TERMINA AQUI ---

\begin{document}
% O conteúdo do seu livro vai aqui...
\end{document}

Dica de Especialista

Mantenha seu preâmbulo organizado. Adicione comentários explicando o propósito de cada pacote. Isso torna seu documento mais fácil de manter.

Módulo 2: Formatando Conteúdo e Estrutura

Aprenda a organizar suas ideias de forma lógica e hierárquica usando seções, listas e ênfases, dando ao seu texto uma estrutura clara e profissional.

Ênfase Semântica

Use \emph{...} para dar ênfase. A vantagem é que o LaTeX adapta a formatação ao contexto. Se o texto ao redor já estiver em itálico, \emph usará o formato reto para criar contraste.

\section{Análise de Resultados}
\textit{Este texto está em itálico.} \emph{Este também.}

\textit{Veja a diferença aqui: \emph{o texto interno volta ao normal.}}

Boas Práticas de Estruturação

Evite usar quebras de linha manuais (\\). Para espaçamento, use parágrafos (uma linha em branco no código). Deixe o LaTeX gerenciar o fluxo do texto.

Módulo 3: O Poder da Matemática

A principal razão pela qual muitos escolhem o LaTeX é sua capacidade superior de diagramar expressões matemáticas.

Referenciando Equações e Unidades Físicas

Para artigos científicos, é essencial numerar e referenciar equações. O pacote siunitx oferece uma forma padronizada de formatar números e unidades.

\usepackage{amsmath}
\usepackage{siunitx} % Para unidades físicas

% ... no corpo do documento ...
A aceleração da gravidade é de \SI{9.8}{\meter\per\second\squared}.
A Equação~\ref{eq:relatividade} define a relação entre massa e energia.
\begin{equation}
    E = mc^2
    \label{eq:relatividade}
\end{equation}

Cenário de Uso Real

Em um artigo de engenharia, usar amsmath, \label, \ref e siunitx não é apenas uma conveniência, é um requisito para a publicação.

Módulo 4: Elementos Flutuantes: Figuras e Tabelas

Enriqueça seus documentos com elementos visuais e dados organizados. Aprenda a inserir, legendar e referenciar figuras e tabelas.

Controlando Flutuantes e Tabelas Flexíveis

O LaTeX posiciona figuras e tabelas onde julga que o layout ficará melhor. Para tabelas que ocupam toda a largura, o pacote tabularx é a ferramenta ideal.

\usepackage{graphicx}
\usepackage{booktabs}
\usepackage{tabularx}

\begin{table}[ht]
    \centering
    \caption{Tabela com coluna de texto flexível.}
    \label{tab:dados}
    \begin{tabularx}{\textwidth}{l X r}
        \toprule
        ID & Descrição & Valor \\
        \midrule
        01 & Descrição longa que pode ocupar múltiplas linhas. & 150.7 \\
        \bottomrule
    \end{tabularx}
\end{table}

Dica de Especialista: Posição do `label`

Sempre coloque o comando \label depois do comando \caption. O \label se associa ao último contador incrementado. Se vier antes, pode se referir à seção em vez da figura/tabela.

Módulo 5: Gerenciamento de Bibliografia com BibTeX

Automatize suas citações e referências. Com BibTeX, você cria um banco de dados de referências e o LaTeX cuida da formatação.

BibTeX vs. BibLaTeX

BibTeX é o sistema clássico. Para necessidades mais complexas (múltiplas bibliografias, customização avançada), o sistema mais moderno BibLaTeX oferece maior flexibilidade.

% Em referencias.bib
@misc{overleaf_site,
    author = {{Overleaf}},
    title = {Overleaf, The Online LaTeX Editor},
    year = {2025},
    howpublished = {\url{https://www.overleaf.com}},
    note = {Acessado em 03-09-2025}
}

Cenário de Uso Real: Uma Tese de Doutorado

Use um gerenciador de referências (Zotero, Mendeley) para criar seu arquivo .bib. No LaTeX, você pode mudar o estilo de citação (ABNT, APA, IEEE) alterando apenas uma linha de código.

Módulo 6: Padrões Acadêmicos Brasileiros com abnTeX2

O ecossistema abnTeX2 é a solução definitiva para criar trabalhos acadêmicos no padrão da ABNT, automatizando capa, sumário e referências.

Além da Estrutura: Listas e Citações Longas

O abnTeX2 gera a Lista de Figuras/Tabelas. Para citações com mais de 3 linhas, o ambiente citacao aplica o recuo e a fonte exigidos pela ABNT.

% ...
\begin{document}
% ...
\listoffigures
\listoftables
\tableofcontents

\textual
\chapter{Revisão da Literatura}
Conforme destaca Knuth \cite[p. 5]{knuth1984}:
\begin{citacao}
A principal vantagem do TeX é sua capacidade de produzir
documentos de alta qualidade tipográfica.
\end{citacao}
% ...
\end{document}

Módulo 7: Escalando Projetos e Personalização

Para documentos longos, organize o projeto em múltiplos arquivos. Criar seus próprios comandos pode economizar tempo e aumentar a consistência.

\input vs. \include e Ambientes Customizados

\input é um "copia e cola". \include é mais poderoso: inicia uma nova página e permite compilar apenas capítulos específicos com \includeonly, acelerando a compilação de livros.

% No preâmbulo
\newenvironment{objetivo}
    {\begin{quote}\sffamily\bfseries}
    {\end{quote}}

% No texto
\begin{objetivo}
O objetivo desta seção é demonstrar a criação de
ambientes customizados em LaTeX.
\end{objetivo}

Boas Práticas: Crie seu próprio pacote .sty

Para projetos grandes, mova seu preâmbulo (pacotes, comandos) para um arquivo meuestilo.sty. No documento principal, use \usepackage{meuestilo} para manter o código limpo e portátil.

Módulo 8: Projeto Final - Artigo Acadêmico ABNT

É hora de consolidar o conhecimento. O objetivo é criar um documento acadêmico completo, bem estruturado e seguindo o padrão ABNT.

Dicas para o Sucesso

  • Comece com um template: Use um modelo confiável (como os do Overleaf ou abnTeX2).
  • Compile com frequência: É mais fácil encontrar um erro em 5 linhas novas do que em 5 páginas.
  • Use o controle de versão: O Overleaf possui histórico. Para projetos sérios, conecte-o a um repositório Git.
  • Entenda os warnings: Avisos como "citation undefined" ou "overfull hbox" indicam pontos que podem ser melhorados.
